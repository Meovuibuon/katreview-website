# Screenshots Directory

This directory contains screenshots of the KatReview website for documentation purposes.

## Screenshots to Add:

- `homepage-desktop.png` - Desktop homepage view
- `homepage-mobile.png` - Mobile homepage view  
- `article-page.png` - Individual article page
- `admin-panel.png` - Admin interface
- `search-feature.png` - Search functionality demo

## How to Take Screenshots:

1. **Desktop Screenshots**: Use browser dev tools to set viewport to 1920x1080
2. **Mobile Screenshots**: Use browser dev tools mobile view (375x667 or 414x896)
3. **Full Page Screenshots**: Use browser extensions or dev tools
4. **File Format**: PNG format recommended for best quality
